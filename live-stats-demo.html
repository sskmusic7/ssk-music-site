SSK Music Live Stats

Real-time YouTube & Spotify streaming numbers

🎯 About This Demo

This is the live stats component for SSK Music Beats. It displays real-time streaming numbers from YouTube and Spotify with animated counters and embedded playlist modals.

✨ Features

  • Real-time updates - Polls APIs every 10 seconds
  • Smooth animations - Stock market ticker style counting
  • Live indicators - Pulsing dots show active polling
  • Interactive modals - Click cards to view playlists
  • Responsive design - Works on all devices
  • Demo mode - Works without API keys for testing

🔑 Setup Required

To connect live data, configure these in the script:

  • YOUTUBE_API_KEY - Get from Google Cloud Console
  • YOUTUBE_PLAYLIST_ID - Your YouTube playlist ID
  • SPOTIFY_CLIENT_ID - Get from Spotify Developer Dashboard
  • SPOTIFY_CLIENT_SECRET - Your Spotify app secret
  • SPOTIFY_PLAYLIST_URI - Your Spotify playlist URI
